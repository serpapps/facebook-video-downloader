# Facebook Downloader (Browser Extension)

> Download Facebook videos, images, Reels, and post text in one browser workflow.

Facebook Downloader is a browser extension built for users who want a cleaner way to save Facebook content without using external downloader sites. It detects visible videos, images, Reels, and post text directly in your browser session and lets you save them in standard formats for offline reference.

- Save Facebook videos and Reels as MP4 files
- Download images from posts and albums
- Export visible post text for notes or archiving
- Bulk-download visible media from the current page
- Keep local copies of content you can already access in your session

## Why Facebook Downloader

Facebook is built for scrolling, sharing, and in-platform viewing, not for straightforward downloads. Videos cannot usually be saved with a right-click, image posts are mixed into dynamic layouts, and post text is awkward to preserve cleanly if you are building a personal archive or research set.

Facebook Downloader is built for that exact workflow. It scans visible Facebook content, groups the media by type, and gives you a direct way to save the content you already have access to without leaving the browser.

## Features

- Video detection for feed posts, watch pages, profile content, and Reels
- Image downloads for visible post media and supported albums
- Post-text export for visible Facebook posts
- In-page download controls on supported media
- Bulk-download workflow for visible assets on the page
- MP4 export for supported Facebook videos
- Source-format image saving where available
- Cross-browser support for Chrome, Edge, Brave, Opera, and Firefox

## How It Works

1. Install the extension from the latest release.
2. Open Facebook and navigate to a feed, profile, Reel, watch page, or post.
3. Scroll until the content you want is visible on the page.
4. Use the in-page controls or open the extension popup.
5. Download individual items or save the visible set in bulk.
6. Open the exported files from your Downloads folder.

## Step-by-Step Tutorial: How to Download Content from Facebook

1. Install Facebook Downloader from the latest GitHub release.
2. Sign in to Facebook if the content you want requires login access.
3. Open the page, post, Reel, or profile section you want to save from.
4. Scroll until the target media is fully visible and loaded.
5. Click the extension button or the on-page download control.
6. Review the detected videos, images, and text entries.
7. Download individual items or run a bulk download for visible media.
8. Open the saved MP4, image, or TXT files from your Downloads folder.

## Troubleshooting

**The extension is not finding the video**  
Make sure the post is visible and fully loaded before scanning or clicking the in-page control.

**A Reel or watch-page item is missing**  
Open the media fully and let playback begin before trying again.

**Bulk download missed some posts**  
Scroll further so more posts load, then run another scan.

**The post requires login or restricted access**  
The extension only works on content you can already access in your active Facebook session.

**The text export looks incomplete**  
Expand any truncated post content before scanning.
